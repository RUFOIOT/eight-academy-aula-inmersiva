06 · Próximos Pasos

Antes de firmar cualquier contrato

📋
Solicitar cotizaciones pendientes
A VMP: Versión C ajustada a 70 m² (como indica su propia cotización) o cotización solo de contenido para sistema de 8 proyectores.

A 3LASER: Cotización de instalación real (el $80 es inviable para 8 proyectores), incluir calibración y videomapping.

Local: Cotizaciones de obras civiles y eléctricas a un arquitecto/electricista de confianza.
🤝
Negociar paquete con VMP para el modelo híbrido
VMP ya indica que desde la 2da aula el precio baja ~15%. Si Eight Academy tiene intención de expandir a 3–5 aulas, proponer contrato marco desde el inicio. Esto reduce el fee mensual de contenido y puede incluir descuento en asesoría técnica.

Negociar que VMP optimice el contenido para 8 proyectores (no solo para su configuración de 4).
🏗️
Definir presupuesto fijo para obras
Las obras civiles y eléctricas son la mayor variable de costo impredecible. Antes de arrancar, visita técnica de un electricista para confirmar si el tablero actual soporta la carga o requiere ampliación.

Recomendación: Reservar $6,000 como presupuesto fijo para obras (civiles + eléctricas + audio), suficiente para la mayoría de casos en 70 m².
